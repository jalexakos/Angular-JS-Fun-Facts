What Are JavaScript Promises?

According to MDN, a JavaScript Promise is a proxy for a value not necessarily known when the promise is created. It allows you to associate handlers with an asynchronous action's eventual success value or failure reason. In plain English, it essentially means that JavaScript can supply data at a later point based on some sort of "event", which allows webpages to be more dynamic in their uses.
