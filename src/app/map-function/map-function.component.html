What Does The Map Function Do?

The map function in JavaScript allows you to call a function on an array of elements and returns an array of the values that result from that called function on each value in the original array. For example, if you passed in an array [1,2,3,4] and then used the map function on that array with a callback function that doubles the passed value, you would return an array of [2,4,6,8]. It is extremely useful for performing the same operation on multiple elements.
