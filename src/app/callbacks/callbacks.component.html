What Are JavaScript Callbacks?

A JavaScript Callback is a function that is executed asynchronously, or non-sequentially. Essentially, a callback function is one that is passed into another function, to be executed at another time based on certain conditions.

Callbacks are important because, again, JavaScript is an event driven language. Thus, we may need code that only fires off after certain events have occurred (i.e. a webpage visitor hovers over an image). If we want that to happen only under the condition that someone hovers over the image, we may have to use a callback.
